{% trans "Conditional Parts" %}

You can make part of wiki page visible just to some users based on user name or group name.

The block starts with the tag [if condition] and ends with tag [endif]. If the condition is true, the block are output.

Possible conditions:

  • [if user in username...] is true if the user has one of the username(s)
  • [if group in group...] is true if the user is in one of the group(s)
